   py-3 sm:py-2              (touch-friendly height on mobile)

2. RESPONSIVE GRID
   grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3
   (1 column mobile, 2 tablet, 3 desktop)

3. RESPONSIVE FLEX
   flex flex-col sm:flex-row   (stack mobile, row desktop)
   flex flex-col sm:flex-row sm:items-center sm:justify-between
   (stacked, then aligned horizontal on desktop)

4. RESPONSIVE TEXT
   text-2xl sm:text-3xl md:text-4xl font-bold
   (smaller mobile, larger desktop)

5. RESPONSIVE GAPS
   gap-2 sm:gap-4 md:gap-6    (tight mobile, loose desktop)
   space-y-2 sm:space-y-4     (vertical spacing)

6. HIDE/SHOW ELEMENTS
   hidden sm:block            (hide mobile, show sm+)
   block sm:hidden            (show mobile, hide sm+)
